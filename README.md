# todo-list-sequelize

一個簡單的todo list
## 功能
- 可創建自己的帳戶
- 顯示當下使用者的所有todo
- 可以增加、刪除、修改todo
## 使用
1. 複製專案
```
git clone https://github.com/kany102030/todo-list-sequelize.git
```
2. 安裝 node.js 與 npm
- https://nodejs.org/
- https://www.npmjs.com/
3. 安裝MySQL
- https://dev.mysql.com/
設定root的密碼為root
4. 於專案資料夾開啟終端機輸入以下指令安裝相依套件
```
npm install
```
